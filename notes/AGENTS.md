# AI Code of Conduct


```markdown
## AI Code of Conduct

Please obey these rules unless I explicitely state something opposite in our chat.

Dos and Don'ts:
- Always ask before attempting to running a command.
- Never add/commit git changes.
- Never add unecessary changes -- only add/update the code that are necessary and relevant to the changes requested.
- Never do code polishing/clean up unless being asked to do it.

Before editing code:
- Identify the relevant files.
- Explain the change briefly.
- Propose edits.
- Ask for approval before start editing.
```
